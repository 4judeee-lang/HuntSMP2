# HuntSMP2

HuntSMP resource packs and plugin expansions.

## Crimson Blade 1.22.1

- Resource pack: `HuntSMP-Pack-1.22.1.zip`
- Addon plugin: `HuntSMP-Crimson-1.22.1.jar`
- Source archive: `HuntSMP-Crimson-1.22.1-Source.zip`
- Complete bundle: `HuntSMP-Crimson-1.22.1-Complete.zip`
- Resource-pack SHA-1: `9d00bb63ffcc2a17bd30a1f7d0dae4957c0b054a`
- Direct pack URL: `https://github.com/4judeee-lang/HuntSMP2/releases/download/v1.22.1/HuntSMP-Pack-1.22.1.zip`

Install the addon JAR beside `HuntSMP-1.21.1.jar` in the server's `plugins` folder. Configure HuntSMP with the direct pack URL and SHA-1 above.

Use `/hunt altar give Crimson` to receive the altar or `/hunt altar spawn Crimson` to place one directly.
